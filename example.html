多功能挂件介绍

什么是多功能挂件？

多功能挂件（MultiWidget）是一个可拖拽、可自定义的浮动工具组件，它允许您在网页中嵌入各种实用功能，如音乐播放器、视频播放器等，并支持添加您自己的自定义组件。

主要特点：

可拖拽定位： 可以自由拖动挂件到网页的任何位置。
可调整大小： 支持自定义调整组件大小，适应不同需求。
多组件切换： 默认内置音乐、视频等组件，可随时切换使用。
自定义组件： 支持添加自定义URL作为新组件。
记忆功能： 可以记住位置、大小和上次使用的组件。
最小化/固定： 支持最小化和固定功能，不干扰正常浏览。
主题切换： 支持深色/浅色主题切换。

使用方法

基本使用

  1. 引入挂件脚本：<script src="widget.js"></script>
  2. 初始化挂件： 
    // 初始化
MultiWidget.init();
    
    // 可选配置参数
MultiWidget.init({
    defaultWidget: 'music',  // 默认显示的组件
    minimized: true,         // 默认最小化
    position: {              // 默认位置
        right: '20px',
        bottom: '20px'
    }
});

配置选项

选项 类型 默认值 说明
defaultWidget String 'music' 默认显示的组件ID
minimized Boolean true 初始是否为最小化状态
position Object {right:'20px', bottom:'20px'} 初始位置设置

内置接口说明

  • 显示挂件： MultiWidget.show();
  • 隐藏挂件： MultiWidget.hide();
  • 最小化： MultiWidget.toggleMinimize();
  • 固定： MultiWidget.togglePin();
